== Description ==

Plugin for create sorted authors list in WordPress blog.

Original authors: Alexey Balin [http://www.masterit.ru/wp-plugin-authors-list/]

== Installation ==


	1. Simply download and unzip it.
	2. Put it in your plugins directory.
	3. Activate it.
	4. Create authors list template [ see example: http://www.masterit.ru/wp-content/uploads/2011/07/authors.php_1.txt ]
	5. Add page with template Authors
